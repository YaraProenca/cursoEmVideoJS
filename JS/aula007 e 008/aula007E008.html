Operadores Aritiméticos

Utilizados para fazer cáculos
  • + => SOMA: 5+2 = 7
  • - => SUBTRAÇÃO: 5-2 = 3
  • * => MULTIPLICAÇÃO: 5*2 = 10
  • / => DIVISÃO: 5/2 = 2,5
  • % => RESTO: 5%2 = 1
  • ** => POTÊNCIA: 5² > 5*5 = 25
